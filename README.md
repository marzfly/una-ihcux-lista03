## 🏎 1° HEURÍSTICA DE NIELSEN
O sistema informa o status de carregamento. Isso se alinha à primeira heurística de Nielsen, que afirma que o sistema deve sempre manter o usuário informado sobre o que está acontecendo de forma clara e entendível. 

## 🍶 PERGUNTA DE REFLEXÃO
"Se o programa demorasse 10 segundos para finalizar, mas ficasse com a tela totalmente parada (em branco), sem as mensagens de '[PROCESSANDO]', o que o usuário provavelmente pensaria? Como a visibilidade de status melhora a Experiência do Usuário (UX)?"
*R: O usuário pensaria que o sistema não está funcionando. Isso quer dizer que a falta de avisos claros atrapalha a experiência do usuário, e, portanto, dificulta o uso, muitas vezes fazendo com que o usuário desista de usar.*
